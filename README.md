# santiagovallejo.com

Sitio estatico tipo bitacora construido con Hugo.

## Requisito unico

Instalar Hugo (una sola vez):

```bash
brew install hugo
```

## Arranque local

Desde esta carpeta, ejecuta:

```bash
hugo server -D
```

Luego abre `http://localhost:1313`.

## Publicacion (Cloudflare Pages)

- Framework preset: `Hugo`
- Build command: `hugo --minify`
- Output directory: `public`
- Version recomendada de Hugo: `0.125+`
